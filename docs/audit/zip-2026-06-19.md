## Findings & état

| ID | Sév. | Description | État |

| Anti-triche | 🟢 Bon | `validate` durci robuste (non-tableau, non-objets, hors-grille, doublons, sur-longueur, sans crash) ; pas de fuite de solution ; engine/server jamais côté front. | ✅ |
| ZIP-D-7 | ℹ️ | Beaucoup de points de passage sans murs (puzzles « reliez les points »). | ⏳ **Acté** : murs en génération = amélioration future. |
