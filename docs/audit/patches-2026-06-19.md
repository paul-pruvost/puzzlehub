## Preuve de correction du validateur
Sous `Σvalue = N²` (assertée) : board complet ⇒ `Σcells = N²` ; gardes `cells ≤ value` ⇒ égalité partout ⇒ `cells = value` ; `bbox ≤ value = cells ≤ bbox` ⇒ rectangle plein ; garde « pas d'autre indice dans la bbox » ⇒ un indice/rectangle. Donc `valid` ⇒ vraie solution Shikaku.
