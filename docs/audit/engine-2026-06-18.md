## Tests ajoutés (anti-triche prioritaire)
- Tango : board mauvaise taille → `invalid/shape` ; valeur hors `{0,1,null}` → `invalid/shape`.
- Queens : colonne hors-grille (`99`) → `invalid/shape` ; mauvaise longueur → `invalid` ; « toute solution pleine validée = unique solution ».
- Total moteur : **18 tests verts**.

## Reste à faire (backlog)

- Property-based (fast-check) : 100+ seeds × 3 difficultés, `countSolutions==1` + `validate(solve)==valid`.
- Mesurer taux d'échec `generate` Queens (BUG-06) lors du spike L1.

## Verdict
Moteur **fiable pour la banque offline** et **sûr comme validateur autoritatif** après correction de BUG-03 et BUG-07. Reste la dette timeout/property-based avant d'attaquer les jeux #P-difficiles (Zip/Patches).
